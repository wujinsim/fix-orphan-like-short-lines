# Privacy Policy

Last updated: May 18, 2026

Fix Orphan-like Short Lines is a plugin for Adobe InDesign developed by Sim Wujin / mulgogi.

This Privacy Policy explains how the plugin handles user data.

## Summary

Fix Orphan-like Short Lines does not collect, store, transmit, sell, or share personal data.

The plugin works locally inside Adobe InDesign.

## Data collection

The plugin does not collect personal information.

It does not collect:

- Names
- Email addresses
- Payment information
- Account information
- IP addresses
- Location data
- Device identifiers
- Analytics data
- Adobe document contents
- Selected text
- InDesign file contents

## Document contents

The plugin analyzes text frames inside Adobe InDesign only to find short lines and help adjust tracking.

All processing is done locally on the user's device.

Document contents, selected text, search results, or layout data are not sent to any external server.

## Local storage

The plugin may store user preferences locally on the user's device, such as:

- Language setting
- Automatic tracking allowance
- Saved ending-mark filters

These settings are stored locally for plugin functionality only.

They are not transmitted to the developer or to any third party.

## Third-party services

The plugin does not use external APIs, analytics services, advertising services, or tracking services.

## Data sharing

The plugin does not share user data with the developer, Adobe, or any third party.

Please note that Adobe may process information separately through Adobe Creative Cloud, Adobe InDesign, Adobe Marketplace, or related Adobe services. Adobe's own data practices are governed by Adobe's privacy policies and terms.

## Children's privacy

The plugin is not directed to children and does not knowingly collect personal information from children.

## Changes to this policy

This Privacy Policy may be updated from time to time.

If changes are made, the updated version will be posted on this page.
